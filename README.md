# pysqm

http://www.unihedron.com/projects/darksky/cd/SQM-LU/readsqmlu.py sayfasındaki yazılım temel alınarak hazırlanmış bir projedir.
Proje zaman içinde geliştirilerek yayınlanmaya devam edecektir.
